接続設定ダイアログ + 実 AI チャット。LM Studio / Ollama / OpenAI / Claude / Gemini / Grok / その他 を OpenAI 互換として切り替え、 GET /models で一覧取得・接続テストし、そのまま会話できます (POST /chat/completions を SSE streaming、応答は ui_md_pre で Markdown 整形)。 create_ui_dialog(controlled・width 指定)、ui_tabs(pill で択一)、 create_ui_scroll_pane(下端追従)、ui_textarea + focus_when の組み合わせ例。 設定は localStorage に保存しますが API キーは保存しません。 ブラウザ直 fetch のため一覧取得・会話は CORS の影響を受けます (UI の表示・操作は接続先が無くても確認できます)。下ペインに実コードを表示しています。
