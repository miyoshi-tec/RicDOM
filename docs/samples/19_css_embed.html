埋め込み (css_for) — RicDOM

create_ui_page で包めない既存アプリの一角に、css_for + make_css_vars の 3 点セットで RicUI の島を作る。埋め込みの公式ルート。

これは RicDOM 公式のサンプルページです。 JavaScript を有効にすると、ナビゲーションバーと実動するデモが表示されます。

既存アプリへの埋め込み。通常は create_ui_page でアプリ全体を 包むのが canon ですが(他のサンプル参照)、create_ui_page で包めない・ 包みたくない場面(既存アプリの一角、iframe、外部ウィジェットホスト等)では css_for + make_css_vars の 3 点セットで RicUI の「島」を 自分で作れます(v0.3.34〜)。下の「既存アプリ」風の領域(RicUI と無関係な自前 CSS)の 中に、テーマの異なる 2 つの島(teal / dark)を埋め込んでいます。 島には portal 系(popup/dialog/toast/tooltip)は置けません (詳細はコード中のコメント参照)。下ペインに実コードを表示しています。
