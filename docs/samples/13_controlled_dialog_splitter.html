create_ui_dialog と create_ui_splitter の controlled mode を使うと、開閉状態を state として扱えるようになり、 他の UI との同期が宣言的に書けます。 このサンプルでは、2 つのチェックボックスと bottom スプリッターの双方向同期、 およびボタンから state 経由で開くダイアログを示します。 下ペインに実際のコードを表示しています（下部 splitter で高さ調整可）。
