RicDOM

JSON で書く、お手軽 リアルタイムコントロール DOM ライブラリ

Electron アプリ・社内ツール・パラメータ調整 UI 向けの JavaScript ライブラリ。 { tag: 'div', ctx: [...] } 形式の JSON で UI を宣言し、 Proxy ベースの自動再描画で動かします。 Virtual DOM 無し、ビルド不要、 <script> タグ 1 つで動きます。

主な特徴

  • JSON 記述 — UI を plain JavaScript object で定義
  • Virtual DOM なし — JSON 差分から実 DOM を直接パッチ。input フォーカス・IME 変換を壊さない
  • Proxy 自動追跡 — s.count = 5 で自動再描画
  • 極小バンドル — RicDOM コア 10KB / RicUI 65KB (minified)
  • テーマ × 密度 — CSS 変数で 5 テーマ × 3 密度
  • Electron / ブラウザ両対応

ハマるアプリ

Electron アプリ、社内業務ツール、プロトタイプ、データ可視化ツール、 パラメータ調整 UI、リアルタイムダッシュボード、IoT デバイスの設定 UI。

苦手なアプリ

大規模 SPA、SEO 必須のマーケティングサイト。

クイックスタート

<script src="RicDOM.min.js"></script>
<div id="app"></div>
<script>
const { create_RicDOM } = RicDOM;
create_RicDOM('#app', {
  count: 0,
  render: (s) => ({
    tag: 'button',
    ctx: [`Count: ${s.count}`],
    onclick: () => { s.count++; }
  })
});
</script>
